## 📖 Overview

**QuickCraft** is a Quality of Life (QoL) modification for **Vintage Story**.

It automates crafting by moving items into the crafting grid and allowing instant bulk crafting with simple controls.

---

## ⚡ How It Works

### Method 1: Survival Handbook (H-Key)

1. Press **H**
2. Open a recipe
3. Use QuickCraft buttons to auto-fill the grid

### Method 2: Crafting Grid

1. Place a recipe item in the grid
2. Use QuickCraft buttons to craft instantly

---

## 🛠️ Buttons

| Button | Function |
|------|----------|
| + | Add One craft setup |
| * | Max possible for one craft |
| ALL | Craft everything automatically |

---

## 🌟 Features

- 🚀 Fast bulk crafting
- 🔒 Client-side only
- 🧠 Smart recipe handling
- 🧩 Works with vanilla & modded recipes
- 🎨 Clean UI integration
