## 📖 Overview

**QuickCraft** is a highly efficient client-side Quality of Life (QoL) modification for **Vintage Story**. 

Tired of endless repetitive clicking when you need hundreds of planks, cobblestone blocks, or meals? This mod streamlines your entire bulk crafting experience, saving you valuable time and physical effort. It integrates three powerful, intelligent buttons directly into the Survival Handbook and standard crafting grids.

With QuickCraft, bulk crafting isn't a chore; it's instant mastery.

## ⚡ How It Works

QuickCraft makes automation simple and integrated.

### Method 1: The Survival Handbook (H-Key Workflow)

This is the recommended workflow for creating items with multiple steps or complex recipes.

1.  Press **`H`** to open your Survival Handbook.
2.  Navigate to and select the recipe you want to craft.
3.  Observe three new, themed buttons that appear in the recipe window. Use them to instantly fill the grid from your inventory. No manual dragging required!

### Method 2: Standard Crafting Grid

If you prefer using the workbench or your 2x2 grid directly.

1.  Place at least one of the final required ingredients in the output slot of your crafting grid.
2.  Shift-click or Alt-click one of the functional QuickCraft buttons that now appear near the grid.

---

## 🛠️ Button Functionality Reference

QuickCraft adds three distinct control buttons. Learning their functions is key to maximum efficiency.

| Button Icon | Function | Description |
| :--- | :--- | :--- |
| **`+`** | **Add One** | Instantly pulls exactly enough resources from your inventory to prepare the grid for **one** craft cycle. |
| **`*`** | **Max (1 cycle)** | Fills the entire grid with the maximum number of items your inventory can support, preparing it for a single crafting action that produces the maximal amount. |
| **`ALL`** | **Craft All** | The definitive bulk option. QuickCraft calculates the maximum items possible, automatically handles all necessary sub-crafts (like turning logs into planks before making a table), and crafts the final items immediately, clearing your bags of all source materials. |

## 🌑 Key Features & Benefits

* 🚀 **Maximum Efficiency:** Reduces hours of tedious clicking and inventory management to seconds.
* 🔒 **Client-Side Only:** This is purely an interface enhancement. It works on any server without requiring the mod to be installed on the server side. Safe for multiplayer.
* 🧠 **Smart Dependency Handling:** The `Craft All` function is intelligent. It handles intermediate steps, so you can craft from raw logs even if you need planks.
* 🧩 **Mod Compatibility:** Works with standard vanilla recipes and is designed to support custom recipes and items added by other modifications.
* 🎨 **Immersive Aesthetic:** The interface elements are carefully styled to match the authentic copper and stone UI of Vintage Story.

## 📦 Installation & Setup

QuickCraft is extremely easy to set up.

1.  **Download:** Get the latest release `.zip` file from the [Releases tab](https://ВАШЕ_ПОСИЛАННЯ_НА_ЗАВАНТАЖЕННЯ/latest).
2.  **Locate Mods Folder:** Find your Vintage Story data directory. The mods folder is typically at `.../Vintagestory/Mods/`.
3.  **Place File:** Copy the downloaded `.zip` file into your mods folder.
4.  **Launch:** Start the game. The mod will activate automatically. Verify in the in-game 'Mods' menu.

> **Requirement:** QuickCraft does not require any additional API mods to function. It is stand-alone.
